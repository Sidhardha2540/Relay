§ 01 Topology

A single source of truth, not a shared file.

Each agent runs its own MCP shim on stdio; the shim proxies tool calls over HTTP to the local daemon. asyncio.Lock + SQLite WAL serializes every mutation. Two agents can never race — the protocol prevents it.

§ 02 Coordination Console

Four stores. Four consistency models.

A single collapsed model would be wrong: a 409 on a discovery is incorrect; LWW on a decision is catastrophic. Each store gets exactly the semantics it needs — and a documented HTTP code for every conflict path.
